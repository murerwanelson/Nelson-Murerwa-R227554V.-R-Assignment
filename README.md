# Nelson Murerwa R227554V - Markov Chain Assignment

This repository contains an R analysis of several Markov chains as part of the R assignment.

## Contents

- `R/Murerwa_Nelson__R227554V__HASTS 416.R` : Main R script implementing simulations, structural analysis, and plots for 5-, 7-state Markov chains and traffic chain.
- `plots/` : Folder containing all plots generated by the script.

## Description

The assignment covers:

1. **Question A1** – 5-State Markov Chain
   - Structural analysis (recurrent, transient, absorbing, periods)
   - Simulated trajectories
   - Steady-state and unconditional probabilities

2. **Question A2** – 7-State Markov Chain
   - Structural analysis (recurrent classes, periods)
   - Simulated trajectories
   - Limiting probabilities

3. **Question A3** – Traffic Markov Chain
   - Analytical and simulated distributions
   - Step-by-step evolution of traffic states
   - Comparison plots
